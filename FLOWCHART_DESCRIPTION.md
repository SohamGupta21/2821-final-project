## Color Scheme

Each step has a distinct color:

1. **Model Input**: Red (#FF6B6B)
2. **Auto-Detect Adapter**: Teal (#4ECDC4)
3. **Generate Facts**: Light Green (#95E1D3)
4. **Pattern Analysis**: Pink (#F38181)
5. **Rule Generation**: Purple (#AA96DA)
6. **Theory Refinement**: Light Pink (#FCBAD3)
7. **Output Rules**: Light Blue (#C7CEEA)

## Flow Description

### Phase 1: Input & Setup (Top Section)
- **Sequential vertical flow** from top to bottom
- Steps 1 → 2 → 3 connected by downward arrows
- All boxes centered horizontally

### Phase 2: Analysis (Middle Section)
- **Horizontal split** after step 3
- Step 3 branches into two parallel paths:
  - Left: Pattern Analysis (step 4)
  - Right: Rule Generation (step 5)
- Both paths converge downward

### Phase 3: Refinement (Bottom Section)
- **Convergence point**: Arrows from steps 4 and 5 meet
- Flow continues to Theory Refinement (step 6)
- Final step: Output Rules (step 7)

## Additional Elements

### Phase Labels
- Positioned on the left side
- Each phase has a colored background box (dashed border)
- Labels: "Phase 1: Input & Setup", "Phase 2: Analysis", "Phase 3: Refinement"

### Pipeline Details Box
- Positioned on the right side
- Light yellow/cream background (#FFF9E6)
- Contains bullet points describing each phase
- Provides additional context for the pipeline steps

## Technical Details

- **Figure Size**: 14×10 inches
- **Resolution**: 300 DPI (high quality for presentations)
- **Arrow Style**: Black arrows with 2.5pt line width
- **Box Style**: Rounded rectangles with padding
- **Text**: Bold, sans-serif font, centered in boxes

## Usage

When you run `python generate_slide_visualizations.py`, this flowchart is automatically generated and saved as:
- **File**: `figures/pipeline_flowchart.png`
- **Size**: ~300-350 KB
- **Format**: PNG with transparent/white background

The flowchart is ready to insert directly into presentation slides!
